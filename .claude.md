## Webflow Code Components

This is a Webflow Code Components project using React and TypeScript.

### Important Documentation

- See `WEBFLOW_CODE_COMPONENTS_GUIDE.md` for a complete guide on creating and deploying Webflow code components
- This project follows the Webflow code components architecture

### Project Structure

- Component definition files use the `.webflow.tsx` pattern
- Webflow configuration is in `webflow.json`
- Components are located in the `src/` directory

### Key Concepts

- Use `declareComponent` from `@webflow/react` to define components
- Use `props` from `@webflow/data-types` to define configurable properties
- Supported prop types: Text, Rich Text, Image, Link, Variant, Visibility
- Components inherit fonts from the Webflow site automatically
- Run `npx webflow library share` to deploy components to Webflow
